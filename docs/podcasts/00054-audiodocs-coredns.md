## What is CoreDNS?

CoreDNS is a flexible, extensible DNS server that can serve as the Kubernetes cluster DNS. It's a CNCF graduated project and the default DNS server for Kubernetes.

**Key Features**:
- Plugin-based architecture for extensibility
- Native Kubernetes integration
- Service discovery for pods and services
- Support for DNS-based load balancing
