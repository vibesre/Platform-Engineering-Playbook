## What is stern?

stern allows you to tail multiple pods and containers within those pods simultaneously. Output is color-coded for quick visual identification.

**Key Features**:
- Tail logs from multiple pods at once
- Regex-based pod selection
- Color-coded output per pod/container
- Support for all containers in a pod
- JSON log parsing and formatting
