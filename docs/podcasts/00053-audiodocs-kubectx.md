## What is kubectx?

kubectx is a utility to manage and switch between kubectl contexts (clusters). It also includes kubens for switching between Kubernetes namespaces quickly.

**Key Features**:
- Fast context switching with `kubectx <context-name>`
- Interactive fuzzy search with fzf integration
- Namespace switching with `kubens <namespace>`
- Shell completion for bash, zsh, and fish
